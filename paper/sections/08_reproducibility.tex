\section{Reproducibility and certificates}

The census was computed from the repaired Project 18 payload tagged
\[
    \texttt{g900-kernel-admission-metric-repair-v1.0.1}.
\]

The Project 19 audit script
\[
    \texttt{scripts/017\_full\_same\_size\_metric\_sweep.py}
\]
enumerates all \(\binom{30}{5}=142506\) same-size carrier-sign toggles and emits JSON, Markdown, and CSV artifacts.

The primary census artifacts are:
\[
\begin{array}{l}
\texttt{artifacts/json/017\_full\_same\_size\_metric\_sweep.json},\\
\texttt{artifacts/md/017\_full\_same\_size\_metric\_sweep.md},\\
\texttt{artifacts/csv/017\_full\_same\_size\_metric\_sweep.csv}.
\end{array}
\]

The census theorem table is independently checked by
\[
    \texttt{scripts/018\_verify\_full\_census\_claims.py}.
\]
This verifier reads the \(017\) CSV and JSON artifacts, recomputes the odd-triangle class counts and medians, checks the no-expansion condition, verifies the monotone compression gradient, and records SHA-256 hashes of the census artifacts and repaired kernel payload.

The primary verification artifacts are:
\[
\begin{array}{l}
\texttt{artifacts/json/018\_full\_census\_theorem\_verification.json},\\
\texttt{artifacts/md/018\_full\_census\_theorem\_verification.md}.
\end{array}
\]

Earlier supporting audits are preserved as numbered scripts and artifacts \(010\) through \(016\). These include the holonomy return compression audit, odd triangle localization audit, ablation audit, same-size toggle controls, zero-odd metric sweep, and balanced odd-triangle-count gradient audit.

The result should therefore be read as a finite computational theorem with an explicit input payload, explicit enumeration rule, explicit distance rule, preserved output certificates, and an independent theorem-verification artifact.
